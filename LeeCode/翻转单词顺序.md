#翻转单词顺序:  
输入一个英文句子，翻转句子中单词的顺序，但单词内字符的顺序不变。为简单起见，标点符号和普通字母一样处理。例如输入字符串"I am a student. "，则输出"student. a am I"。  

输入: "the sky is blue"  
输出: "blue is sky the"

```python
class Solution3:
    def reverseWords1(self, s):
        list_str = list(s.split())
        n = len(list_str)
        for i in range(n // 2):
            temp = list_str[i]
            list_str[i] = list_str[n-i-1]
            list_str[n-i-1] = temp
        return " ".join(list_str)
```
